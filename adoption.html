Implementation Checklist

  1. Reserve @ meta keys and align teams on conventions.
  2. Build/choose a parser to compile .aion → .sJson.
  3. Adopt storage: text for editing; binary for performance.
  4. Integrate embeddings on insert/update for vector search.
  5. Add validation in CI to keep contracts stable.

Start small: wrap key JSON payloads with Aion semantics and expand.
